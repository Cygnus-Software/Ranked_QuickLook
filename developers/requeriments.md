# Instalation

**Node.js**

```bash
sudo apt-get update && sudo apt-get upgrade -y
```

```bash
sudo apt install nodejs
```

```bash
node -v
```

**npm**
```bash
sudo apt install npm
```

```bash
npm -v
```

**Serverless**

```bash
npm install -g serverless@3.21
```
